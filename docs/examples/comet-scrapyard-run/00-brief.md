# Brief

- Project: `Comet Scrapyard`
- Goal: ship a small browser-first arcade game with quick runs and a strong score-chasing loop
- Audience: nostalgic players who like retro arcade pressure and readable systems
- Core fantasy: steer a small salvage tug, grab drifting metal with a magnet beam, dodge hazards, and cash in before the timer or hull runs out
- Hard constraints:
  - browser first
  - likely 2D top-down presentation
  - keyboard and mouse first
  - polished enough to feel intentional, not placeholder-heavy
- Nice to have:
  - escalating risk-reward zones
  - satisfying cash-in cadence
  - a reason to replay beyond one run
