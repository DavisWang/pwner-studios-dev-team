# Canonical Game Spec

## Core Loop

1. Launch into a debris field.
2. Pull scrap toward the tug with a short-range magnet beam.
3. Avoid fast-moving hazards and overloading the cargo hold.
4. Return to the refinery gate to cash in metal.
5. Buy one between-wave upgrade and repeat until the hull breaks.

## Controls

- `WASD`: thrust and steering
- mouse aim: magnet beam direction
- left click or hold: engage magnet beam
- `Space`: burst brake

## Win And Loss

- The score goal is survival plus cash value, not a hard ending.
- A run ends when hull reaches zero or the player is trapped in a hazard field for too long.

## V1 Scope

- one arena
- three scrap types
- three hazard types
- three upgrades
- one between-wave shop state
- one high-score table stored locally

## Out Of Scope

- campaign progression
- narrative
- multiplayer
- controller support

## Assumptions

- The browser-first profile stays active.
- Keyboard and mouse are sufficient for v1.
